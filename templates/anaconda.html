Setup Anaconda Environment (Debian)

Step 1: Install Anaconda

Prerequisites:

sudo apt-get install libgl1-mesa-glx libegl1-mesa libxrandr2 libxrandr2 libxss1 libxcursor1 libxcomposite1 libasound2 libxi6 libxtst6

Download Installer:

Download anaconda installer here

Verify SHA-256 (Recommended):

Check SHA-256 here

sha256sum /path/filename

Install (Python 3.7):

bash ~/Downloads/Anaconda3-2020.02-Linux-x86_64.sh

Step 2: Basic Commands

Create and Activate Environment:

conda create -n myenv
conda env list
conda activate myenv

Install the Desired Python Version:

conda install python

or you can specify the version

conda install python=3.8

Install Libraries:

we can easily install them

conda install numpy pandas scikitlearn matplotlib

To Remove an Environment:

you need to return to the base environment to remove

conda deactivate
conda remove -n myenv --all
